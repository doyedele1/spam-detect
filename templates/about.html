About SpamML

SpamML is a single-page web application binary-classifier system based on the use of supervised machine learning algorithms and several other data preprocessing techniques. The machine learning based spam detector runs on an asynchronous, non-blocking, multiplexing I/O HTTP web server. The web interface is initially built with HTML, CSS & PHP, before it was refactored in ReactJS which enables the client to communicate with the HTTP server which runs on Flask. The system collects emails from the user through a large textarea where suspicious emails can be pasted, runs the spam detection analysis and alerts the user on the potential of the email being spam or not. The database used was trained with a business email dataset using the machine learning data preprocessing techniques in Python. Over 5000 pre-labelled emails were used to build the classifier. The model uses an adaptation of the Naive Bayes algorithm to compute the probability of the email being spam or ham.

SpamML is open-sourced. Click here to view the GitHub repository.
